# 微信小程序-头脑王者辅助

开发环境：py3

相关依赖：requests,json,mitmproxy,urllib

因为mitmproxy不支持windows,所以在 windows下运行 mitmdump -s king.py -p 8888,在安装好mitmproxy证书后，用手机连接pc端同一个wifi，配置好代理（ip：本机ip，端口：8888），既可。
